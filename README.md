# 🔐 SecurePass - Swift 密码生成器

一个简单的 iOS/macOS 密码生成应用，使用 Swift 和 SwiftUI 构建。

## 📸 预览
(这里放一张你的模拟器截图或真机截图，非常重要！)

## ✨ 功能特点
- 自定义密码长度 (8-32 位)
- 可选字符类型 (大小写、数字、符号)
- 一键复制到剪贴板
- 密码强度实时检测

## 🛠 技术栈
- **语言**: Swift 5.9+
- **框架**: SwiftUI
- **架构**: MVVM (尝试将逻辑分离)
- **最低版本**: iOS 15.0 / macOS 12.0

## 🚀 如何运行
1. 克隆项目：`git clone https://github.com/你的用户名/仓库名.git`
2. 使用 Xcode 15+ 打开 `SecurePass.xcodeproj`
3. 选择模拟器或真机，点击运行 (⌘R)

## 📂 项目结构
- `Models/`: 数据模型
- `Views/`: 界面视图
- `Utilities/`: 密码生成逻辑
